# 📖 Overview

Productivity Dashboard is a modern web application that combines multiple productivity tools into a single elegant interface.

Instead of opening multiple applications for task management, daily planning, goal tracking, timers, weather, and motivation, this dashboard provides everything in one beautifully designed workspace.

The interface is inspired by modern SaaS products such as **Linear**, **Vercel**, **Notion**, **GitHub**, and **Microsoft Loop**, featuring a clean layout, glassmorphism, responsive design, and smooth animations.

# ✨ Features

## 📋 Task Management

- Create new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Mark important tasks
- Persistent storage using LocalStorage

---

## 📅 Daily Planner

- Hourly schedule planner
- Organize the entire day
- Automatically saves every change
- Current hour highlighting

---

## 🎯 Daily Goals

- Add personal goals
- Track completion
- Visual progress bar
- Percentage completion
- Real-time dashboard statistics

---

## 🍅 Pomodoro Timer

- 25-minute work session
- 5-minute short break
- 15-minute long break
- Start / Pause / Reset controls
- Audio notification after session completion

---

## 🌤 Live Weather

- Detects user location
- Displays:

  - Current temperature
  - Weather condition
  - Wind speed
  - Location

- Uses Open-Meteo Weather API

---

## 💬 Daily Motivation

- Random motivational quotes
- Online quote API
- Offline fallback quotes

---

## 🕒 Live Date & Time

- Digital Clock
- Analog Clock
- Current Date
- Real-time updates

---

## 🎨 Modern UI

- Premium SaaS design
- Glassmorphism cards
- Animated backgrounds
- Gradient accents
- Elegant typography
- Smooth hover animations
- Beautiful iconography using Lucide Icons

---

## 🌗 Theme Support

- Light Theme
- Dark Theme
- Animated background videos
- Theme preference saved automatically

---

## 💾 Persistent Storage

The application stores user data locally.

Saved automatically:

- Todo List
- Daily Planner
- Daily Goals
- Theme Preference

No backend required.
